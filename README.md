# 🌐 Personal Portfolio Website (Beginner HTML Project)

A simple, single-file personal portfolio website built with plain HTML and CSS.  
No frameworks. No tools. Just one file — open it in a browser and it works.

---

## 🎯 What You Will Learn

- What HTML is and how a webpage is structured
- How CSS makes a page look nice
- Why having a personal portfolio website matters for your career
- How to host it online for **free** using GitHub Pages

---

## 💡 Why a Portfolio Website?

Your CV is a document — but a portfolio website is a **living proof** of your skills.

- Employers can click a link instead of reading a PDF
- You can show real projects, not just describe them
- It sets you apart from other candidates
- It stays up-to-date as you grow

A simple website like this one is enough to make a great first impression.

---

## 📁 Project Structure

```
your-username.github.io/
└── index.html        ← the entire website (HTML + CSS in one file)
```

That's it. One file.

---

## 🚀 How to Put This Online (Step by Step)

1. **Create a free account** at [github.com](https://github.com)
2. **Create a new repository** and name it exactly: `your-username.github.io`
3. **Upload** `index.html` — rename it to `index.html` if needed
4. Wait 1–2 minutes
5. Your site is live at `https://your-username.github.io` 🎉

Paste that link into your CV and anyone in the world can view your portfolio.

---

## ✏️ How to Customise It

Open `index.html` in any text editor (Notepad, VS Code, etc.) and change:

- Your **name** and **bio** at the top
- Your **email** and **GitHub** link
- Each **project** — copy and paste the project block to add more:

```html
<div class="project">
  <h3>Your Project Name</h3>
  <p>What it does.</p>
  <p><strong>Tools:</strong> Tool1, Tool2</p>
</div>
```

---

## 🧱 How the Code Works (Beginner Explanation)

```html
<!DOCTYPE html>      ← tells the browser this is a webpage
<html>               ← start of the page
<head>               ← settings and styles (not visible on page)
  <style> ... </style>  ← CSS: controls colours, fonts, layout
</head>
<body>               ← everything visible on the page goes here
  <h1> ... </h1>     ← big heading
  <p> ... </p>       ← paragraph of text
  <div> ... </div>   ← a box to group things together
</body>
</html>              ← end of the page
```
